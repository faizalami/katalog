Mengapa Alat ini Dibuat?

Indonesia merupakan salah satu negara yang menganut sistem Demokrasi. Hal ini di tandai dengan diadakannya pemilihan Presiden dan wakil Presiden setiap periode-nya. Dan pada tahun 2019 mendatang akan diadakan pemilihan Presiden dan wakil Presiden. Seorang tokoh politik yang ingin mengajukan diri sebagai calon Presiden dan wakil Presiden tentu akan melihat atau mempertimbangkan popularitas mereka berdasarkan opini masyarakat. Banyak sekali cara untuk menentukan popularitas atau kecenderungan masyarakat, salah satunya adalah dengan survei.



Disisi lain, pada era modern ini, masyarakat menggunakan media sosial untuk mengekspresikan pendapatnya mengenai suatu isu khususnya mengenai pemilu 2019. Untuk mengetahui sentimen suatu opini masayarakat tentang opini positif atau negatif pada twitter sangat susah dilakukan dikarenakan perbedaan bahasa, struktur kalimat, dan teknik penyampaian opini serta memiliki bahasa yang bebas atau language exspresion. Hal ini terbukti dengan banyaknya kata tidak baku, karakter aneh dan kata singkatan. Selain itu, banyaknya data tweet opini masyarakat tentang Pemilu 2019, menyebabkan proses analisis menjadi lebih lama dan rumit. Untuk itu perlu adanya proses Opinion Mining untuk memahami, mengekstrak dan mengolah data tekstual secara otomatis kemudian menampilkannya kedalam visualiasa data yang menarik untuk mendapatkan informasi sentimen yang terkandung dalam suatu kalimat opini.

Manfaat dan Kegunaan

Proyek ini mengajukan suatu penelitian untuk melakukan analisis opini masyarakat terhadap calon pada Pemilu 2019 dengan cara opinion mining pada media sosial Twitter berbasis sentiment analisis. Penelitian ini memanfaatkan opini masyarakat terhadap Pemilu 2019, kemudian melakukan mining pada opini-opini tersebut dengan menganalisa sentiment pada opini dan memvisualisasikan hasil opinion tersebut secara menarik. Analisa yang dihasilkan oleh sistem dapat membantu lembaga-lembaga survei yang ada terutama juga dapat menjadi pengetahuan untuk masyarakat terhadap seorang calon pada Pemilu 2019.

Dengan adanya sistem perangkat lunak yang mampu mengklasifikasikan opini maka proses klasifikasi opini dalam jumlah yang besar dapat dilakukan secara komputerisasi sebagai ganti dari proses klasifikasi manual. Hal ini terutama dirasakan manfaatnya untuk lembaga-lembaga survey dalam kaitannya respon masyarakat terhadap Pemilu 2019. Kita dapat mengetahui kecenderungan masyarakat terhadap calon Pemilu 2019 dan menentukan apakah mereka dinilai positif atau negatif di media sosial Twitter.

Fitur

  • Crawling data komentar :
  • Twitter
  • DroneEmprit.id
  • Preprocessing data :
  • Menghapus url pada komentar
  • Menghapus karakter khusus
  • Membersihkan angka-angka
  • Menghapus simbol tanda baca
  • Lower Case
  • Tokenizing
  • Menghapus kata stopword
  • Stemming
  • Opinion Mining :
  • Pembobotan kata menggunakan metode TF-IDF
  • Sentiment komentar menggunakan metode Naïve Bayes Classifier
  • Visualisasi data dan analisa:
  • Visualisasi perbandingan sentiment
  • Analisa perkembangan tren sentiment sebelum pemilu
  • Analisa perkembangan tren sentiment setelah pemilu
  • Menghitung nilai akurasi
  • Menghitung nilai presisi
  • Menghitung nilai recall

Metode / Desain Sistem

  • Input :
  • data tweet yang sudah disediakan oleh web academic.droneemprit.id (Data training) dan Twitter (Data Testing) untuk beberapa query pencarian yang diinginkan dalam penelitian ini yaitu query tentang Prabowo, Jokowi, Sandiaga Uno, Maruf Amin.
  • Proses :
  • Preprocessing data DroneEmprit atau penghapusan data-data yang tidak perlu
  • Selection fitur menggunakan metode TF-IDF
  • Proses opinion mining untuk menganalisa nilai sentiment menggunakan metode naïve bayes classifier
  • Hasil model kemudian di testing pada data twitter
  • Output :
  • Hasil opinion mining berupa jumlah perbandingan nilai dan tren sentiment dari setiap query yang ada kemudian di visualisasikan kedalam tool kibana.
